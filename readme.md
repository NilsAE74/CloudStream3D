# CloudStream3D

CloudStream3D is a desktop application for viewing large XYZ point clouds using GPU-accelerated rendering.

## Features
- XYZ file loading
- RGB support
- 3D navigation (orbit, pan, zoom)
- Electron-based Windows application
- Builds to standalone `.exe`

## Tech Stack
- Electron
- Three.js
- Node.js
